Laundry Mart Website
This is a basic website for a laundry business. I made it using HTML to show what services are offered, how much they cost, and how customers can get in touch.

What’s on the page?
About Us: A little intro about why we love clean clothes.

Our Services: A list of things we do (washing, ironing, etc.).

Price List: A simple table so people know what to pay.

Booking Form: A place for customers to type in their name and phone number.

How to use it
Save the code as index.html.

Double-click the file to open it in your web browser.

That's it!
